

# About the project...................................

A simple blog web application built using React, React Router, and JSONPlaceholder Fake API.
This project includes blog listing, searching, viewing Author details etc.

# Features
•	Fetch all blogs from Fake API
•	Display blog list with title and author
•	detailed blog page
•	Comments section
•	User profile fetching

# Search Features
•	Search by blog title
•	Search by author name
•	Search bar inside Navbar
•	Shows placeholder popup text in search box 
# Routing
•	/ – Home
•	/blogs – Blog List
•	/blogs/:id – Single Blog Page
•	/users – Users List
•	/users/:id – User Details
